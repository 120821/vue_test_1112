# vue_test_20231112
1. 创建项目
```
vue create vue_test_20231112
```

2. 进入项目目录
```
cd my-project
```
3. 启动服务
```
npm run serve
```
4. 安装vue-router
```
npm install vue-router
```
使用：
```
import Vue from 'vue'
import VueRouter from 'vue-router'

Vue.use(VueRouter)
```
